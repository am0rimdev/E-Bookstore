# E-Bookstore

## Vinícius Alves Amorim


### Website para venda e leitura de livros digitais
Este é um website voltado para a venda de livros digitais em formatos como PDF, EPUB e MOBI por exemplo, no qual possui reviews de youtubers sobre os livros, link para versões físicas dos livros na Amazon e um leitor de ebook integrado com suporte a vários formatos.
